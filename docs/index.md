### Getting Started

1. **Run:** `go build -o darkstation main.go && ./darkstation` (or `make build` then `./darkstation`)
2. **Dev start deck:** `./darkstation -level N` or `LEVEL=N go run .`
3. **Test:** `make test` — **Lint:** `make codestyle`
4. **Translations:** Edit `po/default.pot`, run `make mo`

For full setup and tasks see [development-guide.md](./development-guide.md) and the project [README](../README.md).
